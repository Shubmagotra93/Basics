Git :  Distributed Version control System
Git hub : Online platform to manage repositries

Steps: 
	1. Download Git and check git --version
	2. Add Project to git(Create files/folders)
	3. Git Commands:

		git config --global user.email "shubmagotra93@gmail.com"
		git config --global user.name "shubmagotra93" 

		a) git init
		b) git status
		c) git add
		d) git commit -m "add message here"
		e) git remote add origin https://github.com/Shubmagotra93/Repo2.git
		f) git push -u origin master
